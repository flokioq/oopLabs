# Лабораторные работы по ООП (C++)

Проект содержит 7 лабораторных работ, каждая реализована в отдельном namespace.

## Сборка
1. Откройте `OOP_Labs.sln` в Visual Studio.
2. Выберите конфигурацию: **Debug | x64**.
3. Нажмите **Сборка → Собрать решение**.
4. Запустите через ▶️.

> Требуется Visual Studio 2019/2022/2025.

## Описание лаб

### Лаба 1 (Namespace A)
Работа с вектором: сложение, норма, изменение компонент, вывод.

### Лаба 2 (Namespace B)
Односвязный список символов. Ввод строки → вывод в обратном порядке.

### Лаба 3 (Namespace C)
Поиск элементов последовательности, не входящих в заданный интервал.

### Лаба 4 (Namespace D)
Наследование: базовый класс `VECTOR` и производный `VECTOR_EXT`.

### Лаба 5 (Namespace E)
Перегрузка операторов: `+`, `<<`, `>>` для вектора.

### Лаба 6 (Namespace F)
Шаблонный класс для работы с целыми и комплексными числами.

### Лаба 7 (Namespace G)
Работа с файлами и форматированный вывод.
